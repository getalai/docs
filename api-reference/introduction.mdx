# How to get your API Key

Before you can use the Alai API, you'll need to generate an API key:

1. **Sign up** at [app.getalai.com](https://app.getalai.com)
2. **Click on your name** in the left sidebar
3. **Select "API"** from the dropdown menu
4. **Click "Add new API key"** to generate your key
5. **Copy and securely store** your API key - you'll need it for authentication

<Warning>
  Keep your API key secure and never share it publicly. Treat it like a
  password.
</Warning>

## Quick Start

Generate a presentation with a single API call:

```bash
curl --request POST \
  --url https://slides-api.getalai.com/api/v1/generations \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: multipart/form-data' \
  --form 'data={
    "input_text": "Quarterly sales increased 25% YoY. Key drivers: enterprise segment growth, new product launches, expanded market reach.",
    "additional_instructions": "Make it infographic",
    "presentation_options": {
      "title": "Q4 2024 Sales Report",
      "theme_id": "FLAT_WHITE",
      "slide_range": "6-10",
      "choices_per_slide": 1
    },
    "text_options": {
      "language": "English (US)",
      "tone": "PROFESSIONAL",
      "content_mode": "preserve",
      "amount_mode": "essential"
    },
    "image_options": {
      "include_ai_images": true,
      "style": "three_d"
    }
  }'
```

### Response

```json
{
  "presentation_url": "https://app.getalai.com/presentation/<uuid>",
  "presentation_id": "<uuid>"
}
```

## Request Body Schema

The API accepts a `data` field containing a JSON object with the following structure:

### Root Fields

<ParamField body="input_text" type="string" required>
  The main content that drives your presentation generation. This text forms the
  foundation of all slides created.
</ParamField>

<ParamField body="additional_instructions" type="string">
  Custom generation instructions for AI that fine-tune how content is delivered
  on the presentation.
</ParamField>

### presentation_options

<Expandable title="presentation_options" defaultOpen={false}>
  <ParamField body="title" type="string">
    Sets the title displayed in your presentation.
  </ParamField>

  <ParamField body="theme_id" type="string" default="FLAT_WHITE">
    Defines which theme from Alai will be used for presentation.

    **Options:** `FLAT_WHITE`, `DESERT_BLOOM`, `LAPIS_DAWN`, `EMERALD_FOREST`, `COSMIC_THREAD`, `DONUT`, `OAK`, `OBSIDIAN_FLOW`, `MIDNIGHT_EMBER`, `AURORA_FLUX`

  </ParamField>

  <ParamField body="slide_range" type="string" default="6-10">
    Controls the target number of slides generated from your content.

    **Options:** `auto`, `1`, `2-5`, `6-10`, `11-15`, `16-20`, `21-25`

  </ParamField>

  <ParamField body="choices_per_slide" type="integer" default="1">
    The number of slide variants to generate per slide (1-4). We usually generate 4 variants per slide on the product but the API gives you freedom to choose.

    **Options:** `1`, `2`, `3`, `4`

  </ParamField>
</Expandable>

### text_options

<Expandable title="text_options" defaultOpen={false}>
  <ParamField body="language" type="string" default="English (US)">
    Specifies the language for generated content and AI-generated elements.

    **Options:** `English (US)`, `English (UK)`, `Spanish (Latin America)`, `Spanish (Mexico)`, `Spanish (Spain)`, `French`, `German`, `Italian`, `Portuguese (Brazil)`, `Portuguese (Portugal)`, `Dutch`, `Polish`, `Russian`, `Japanese`, `Korean`, `Chinese (Simplified)`, `Hindi`, `Swedish`, `Norwegian`, `Danish`, `Finnish`, `Greek`, `Turkish`, `Czech`, `Hungarian`, `Romanian`, `Bulgarian`, `Ukrainian`, `Vietnamese`, `Thai`, `Indonesian`

  </ParamField>

  <ParamField body="tone" type="string" default="PROFESSIONAL">
    Defines the writing voice, style, and perspective applied to generated content.

    **Options:** `DEFAULT`, `PROFESSIONAL`, `CASUAL`, `TECHNICAL`, `EDUCATIONAL`, `INSPIRATIONAL`, `NARRATIVE`, `PERSUASIVE`, `AUTHORITATIVE`, `EMPATHETIC`, `CUSTOM`

  </ParamField>

{" "}

<ParamField body="tone_instructions" type="string">
  Custom tone directives when no preset tone matches your needs. **Set only when
  tone is `CUSTOM`.**
</ParamField>

  <ParamField body="content_mode" type="string" default="preserve">
    Controls how input text is processed and adapted for presentation format.

    **Options:** `preserve`, `enhance`, `condense`, `custom`

  </ParamField>

{" "}

<ParamField body="content_instructions" type="string">
  Custom directives for text processing when no preset mode fits. **Set only
  when content_mode is `custom`.**
</ParamField>

  <ParamField body="amount_mode" type="string" default="essential">
    Controls the density of text appearing on each slide.

    **Options:** `minimal`, `essential`, `balanced`, `detailed`, `custom`

  </ParamField>

  <ParamField body="amount_instructions" type="string">
    Custom text density directives for specialized requirements. **Set only when amount_mode is `custom`.**
  </ParamField>
</Expandable>

### image_options

<Expandable title="image_options" defaultOpen={false}>
  <ParamField body="include_ai_images" type="boolean" default="true">
    Whether to generate AI-powered illustrations alongside your content.
  </ParamField>

  <ParamField body="style" type="string" default="three_d">
    Controls the visual style of AI-generated images.

    **Options:** `auto`, `realistic`, `artistic`, `cartoon`, `three_d`, `custom`

  </ParamField>

  <ParamField body="style_instructions" type="string">
    Custom image generation directives for specialized visual requirements. **Set only when style is `custom`.**
  </ParamField>
</Expandable>
